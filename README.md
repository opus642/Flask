# Readme  
这是WEB实训期末作业的后端部分。  
## Install  
这个项目使用了Flask, 请使用以下命令确认您已安装。  
```
$ pip install flask  
```  
## Usage  
输入以下命令：  
```
$ cd flask-tutorial
$ export FLASK_APP=flaskr  
$ export FLASK_ENV=development  
$ flask run -h 0.0.0.0 
```  
在PC端或移动端访问端口为5000的网址即可进行注册，登录，发帖等操作，仅供测试。
